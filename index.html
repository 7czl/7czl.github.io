Qutebrowser

Posted on 2023-03-29 In 生产力与日常使用

一个vimlike broswer
快捷键熟悉后可以脱离鼠标使用。

常用指令：
切tab：KJ
向前向后： HL
滚动：hjkl
关闭tab：d
回滚关闭tab：u
插入模式：输入文字i
点击按钮： f
刷新页面：r/R
快速移动：
gg 开头 G末尾
:help快速打开帮助
:bind查看按键绑定

这个qb是真好用，直接替换掉FF
python的扩展可以简化userscript 有待发掘其功能
糙快猛 :spawn –userscript script_name subcommand

桌面和窗口管理

Posted on 2023-03-29 In 生产力与日常使用

yabai + skhd + statusbar + limelight + stack(optional)

skhd的快捷键设置
移动光标：
option + h / j / k / l
打开终端:
alt + return / ctrl + fn + t
调整窗口布局:
shift + option + h / j / k / l
将当前窗口移动至工作空间:
shift + option + m 最后一个工作区
shift + option + p 前一个工作区
shift + option + n 后一个工作区
shift + option + 1 / 2 / 3 / 4

桌面环境

Posted on 2023-03-29 In 生产力与日常使用

OS X的桌面环境叫Aqua
terminal emulator
终端模拟器 : alacritty
终端复用器：zellij
终端文件管理器:ranger
文件编辑器:neovim

窗口管理器: yabai
shell: zsh

终端和桌面管理器还有shell的区别
WM和DE是高度关联的

对linux来说
基于gtk3:
gnome-shell use Mutter
cinnamon use Muffin
mate use marco
Unity use Compiz
XFCE4 use XFWM4
KDE Plasma5 use kwin_x11 or kwin_wayland
桌面环境的介绍
桌面环境的自定义包括窗口管理器，或混成器，任务栏和应用程序，通常包括终端模拟器，文件管理器和文本编辑器

包含桌面管理与显示服务器与混成器的有:Mutter，Muffin，Compiz
单独的桌面管理器bspwm，i3，Xnomad，dwm
不同桌面管理器支持的布局方式： stacking，Tiling，floating

桌面环境的定制包含哪些内容？
主题，颜色方案，墙纸，文件管理器中的上下文菜单，窗体动画，光标和图标集合。

软件开发中的魔法

Posted on 2023-03-28 Edited on 2023-03-29

什么是软件开发中的魔法
在日常使用的软件或常见的事物，无法理解其工作原理的东西都可以称之为魔法，它们看起来是未知的（至少对你来说是）
未知的东西使我产生焦虑的来源。始终提醒自己软件领域中得魔法在我未理解前看起来都像是魔法，并没有不可知得东西。只是我还不知道而已。
Dave方法：
磁带机的工作原理，收音机是如何工作的，放大和调频模块，立体声FM和单声道FM以及AM的信号差别。然后是数字音频， ADC、DAC、采样
